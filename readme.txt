FAQ Builder is a WordPress plugin that automatically generates FAQ pages from the service PDF manual.

== Description ==

FAQ Builder is an innovative WordPress plugin that creates FAQ pages effortlessly by uploading PDF documents. This plugin uses the OpenAI API to analyze the manual and guides to extract and organize important Q&A, contributing to reduced support costs and improved user experience.

Main Features:
- Automatic FAQ generation from PDF documents
- Intelligent keyword analysis and question extraction
- Editable FAQ sections
- Simple and intuitive interface
- Customizable design options

== Installation ==

1. Install and activate the plugin.
2. Navigate to the FAQ Builder settings page and set the OpenAI API key.
3. Upload a PDF file and generate an FAQ page.
4. Create a new page and insert the [faq_builder] shortcode.
5. Customize the FAQ page design and content if needed.

== Frequently Asked Questions ==

= Is this plugin free? =

Yes, this plugin is free to use. However, you need to have an OpenAI API key to use the plugin, which costs you.

= What type of PDF files are suitable? =

Text-based PDF documents with a clear structure are best suited for this plugin.
